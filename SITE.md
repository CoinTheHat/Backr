# Site Vision: Backr

## 1. Platform Overview
Backr is a decentralized, censorship-resistant membership platform built on the **Mantle Network**. It empowers creators to own their audience, set their own prices, and receive instant payouts without intermediaries.

## 2. Core Value Props
*   **Decentralized:** Smart contract-based memberships.
*   **Zero Monthly Fees:** Flat 5% fee on successful payments only.
*   **Instant Payouts:** Direct settlement to creator wallets.
*   **Ownership:** Creators own their data and audience relationship.

## 3. Sitemap
- [x] Home (`/`) - Landing page with value props.
- [x] Explore (`/explore`) - Creator discovery.
- [x] Dashboard (`/dashboard`) - Creator management studio.
- [x] Community (`/community`) - Member interaction.
- [ ] Profile (`/[creator]`) - Public creator profile page.

## 4. Roadmap
- [ ] **Home Page Refresh:** Improve "friendliness" and visual appeal with new Design System.
- [ ] **Onboarding Flow:** Simplify wallet connection and profile setup.
- [ ] **Profile Customization:** Allow creators to theme their pages.

## 5. Creative Freedom
- Use "Cosmic" imagery (stars, nebulas) but keep it "clean" and "app-like", not "game-like".
- Focus on "Creator Success" visuals.
